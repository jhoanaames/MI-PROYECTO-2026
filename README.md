## 📖 Descripción
Este proyecto consiste en el desarrollo de una página web de una florería, donde los usuarios pueden visualizar diferentes arreglos florales, conocer sus características y explorar los productos disponibles.

La página ha sido diseñada utilizando HTML y CSS, enfocándose en una presentación visual atractiva y organizada.

## 🎯 Objetivo
El objetivo es aplicar conocimientos de desarrollo web básico mediante la creación de una página estática, mejorando habilidades en diseño, estructura y estilos.

## 🛠️ Tecnologías utilizadas
- HTML
- CSS
- GitHub

## ⚙️ Funcionalidades
- Visualización de arreglos florales 🌹
- Diseño estructurado con HTML
- Estilos personalizados con CSS
- Navegación sencilla
- Diseño adaptable (responsive básico)
